# so_long

## Project Overview
A simple 2D tile-based game built using the MiniLibX library.

The player must collect all collectibles before reaching the exit. Before that, exit is blocked.

## Concepts Covered
- Event hooks
- Window management
- Path validation via Flood-fill algorithm
- Texture loading

## ⚙️ Build
```bash
make
```

## Usage

```
./so_long sources/maps/bigmap.ber
```

## Controls

- W / A / S / D or Arrow keys
- ESC to quit

## What I learned

- Graph traversal
- Game loop fundamentals
